\section{Типы МВС}

\begin{frame}{Типы МВС}
Наиболее распространенные типы многопроцессорных вычислительных систем:
\begin{itemize}
    \item Системы высокой надежности,
    \item Системы для высокопроизводительных вычислений,
    \item Многопоточные системы.
\end{itemize}
\end{frame}
